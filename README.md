# bash_system_maintenance_suite
A collection of Bash scripts designed to automate essential system maintenance tasks on Linux. The suite includes scripts for automated system backups, software updates, log monitoring, and cleanup. It features a unified menu-driven interface for easy execution, along with error handling and logging for reliability.
